# 课堂点名器/签到系统 设计文档

**日期**: 2026-03-21
**项目**: Class Roll Call System

## 1. 项目概述

一个基于浏览器的课堂点名器，使用 3D 地球旋转动画作为视觉效果，支持内置学生名单、点名记录、迟到权重管理等功能。

**使用场景**: 课堂环境，老师在教室电脑上使用，投影显示

## 2. 技术选型

| 技术 | 用途 |
|------|------|
| HTML5 | 页面结构 |
| CSS3 | 样式和动画 |
| Vanilla JavaScript | 业务逻辑 |
| Three.js | 3D 地球渲染 |
| IndexedDB | 数据持久化 |

**实现方式**: 混合方案 - 单个 HTML 文件，内部代码按功能模块组织

## 3. 功能需求

### 3.1 核心功能
1. **内置模拟学生名单** - 预设 20 个中文学生姓名
2. **点名记录** - 记录每次点名的时间、学生、状态
3. **迟到权重系统** - 迟到次数越多，被点中概率越高（上限权重 5）
4. **导出记录** - 导出为 CSV 格式
5. **3D 地球动画** - 持续旋转作为背景效果

### 3.2 应用状态

| 状态 | 描述 | UI行为 |
|------|------|--------|
| **IDLE** | 空闲，等待点名 | "开始点名"按钮可用 |
| **SPINNING** | 地球加速旋转选择中 | 按钮禁用 |
| **SHOWING_NAME** | 显示选中学生姓名 | 按钮禁用，状态按钮显示 |
| **AWAITING_STATUS** | 等待选择状态 | 按钮禁用，状态按钮高亮 |

### 3.3 点名流程（详细）

```
1. 老师点击"开始点名"（IDLE → SPINNING）
2. 系统根据权重随机选择学生（后台计算）
3. 地球加速旋转 1.5 秒（rotationSpeed: 0.01）
4. 旋转减速 0.5 秒回到正常速度
5. 显示学生姓名（SHOWING_NAME）
   - 姓名从地球中心放大弹出（scale 0 → 1.2 → 1）
   - 动画时长 600ms，使用 ease-out 缓动
6. 显示状态选择按钮：正常 / 迟到 / 缺勤（AWAITING_STATUS）
7. 老师点击状态按钮
8. 保存记录到 IndexedDB
9. 更新该学生迟到计数（如果迟到）
10. 更新今日统计
11. 姓名显示保持 2 秒后淡出（IDLE）
```

### 3.4 动画时机规范

| 动画 | 触发条件 | 时长 | 效果 |
|------|----------|------|------|
| 地球持续旋转 | 页面加载后 | 无限 | 速度 0.001 |
| 地球加速旋转 | 点击"开始点名" | 1.5s + 0.5s | 先加速后减速 |
| 姓名弹出显示 | 选中学生后 | 600ms | scale 0→1.2→1 |
| 姓名淡出 | 状态确认后 | 2s 延迟 + 500ms | opacity 1→0 |

### 3.5 边界情况处理

- **所有学生已点过** - 允许重复点名，不限制
- **迟到计数上限** - 最高权重限制为 5（迟到8次后不再增加权重）
- **取消状态选择** - 点击姓名区域或 ESC 键取消当前点名
- **同学生重复选中** - 允许，记录追加保存
- **今日统计定义** - 基于日历日期（00:00-23:59）

### 3.6 数据管理

- **清空数据** - 需要二次确认弹窗，防止误操作
- **迟到重置** - 提供"重置迟到计数"按钮，单独重置所有学生的 lateCount
- **查看记录** - 点击"查看记录"打开弹窗，显示所有历史记录

## 4. 数据结构

### 4.1 学生数据 (Student)
```javascript
{
  id: string,           // UUID
  name: string,         // 学生姓名
  lateCount: number,    // 迟到次数（0-8，对应权重1-5）
  createdDate: number   // 添加到系统的时间戳
}
```

### 4.2 点名记录 (Record)
```javascript
{
  id: string,           // UUID
  studentName: string,  // 学生姓名
  timestamp: number,    // 时间戳
  status: 'normal' | 'late' | 'absent'
}
```

## 5. IndexedDB 数据库

```
数据库名: ClassRollCallDB
版本: 1

对象存储 'students':
  - keyPath: 'id'
  - indexes: name, lateCount, createdDate

对象存储 'records':
  - keyPath: 'id'
  - indexes: timestamp, studentName
```

**降级方案**: 如果 IndexedDB 不可用，自动降级使用 localStorage

**版本升级策略**: 版本号变更时清空旧数据重新初始化（简单处理）

## 6. 加权随机算法

```
基础权重 = 1
迟到权重 = min(1 + (lateCount × 0.5), 5)

学生被选中概率 = 该学生权重 / 所有权重之和
```

**示例**:
- 学生 A：迟到 0 次 → 权重 1
- 学生 B：迟到 2 次 → 权重 2
- 学生 C：迟到 8 次 → 权重 5（上限）

总权重 = 8，学生 C 有 62.5% 被选中概率。

## 7. Three.js 地球配置

```javascript
{
  radius: 2,
  normalRotationSpeed: 0.001,
  fastRotationSpeed: 0.01,
  texture: null,  // 使用程序生成网格纹理，无需外部资源
  pointLight: { x: 5, y: 5, z: 5 },
  ambientLight: 0.6,
  backgroundColor: 0x0a0a1a
}
```

**性能优化**:
- 使用程序生成纹理，避免网络加载
- 低性能设备检测：requestAnimationFrame < 30fps 时自动降低渲染质量
- 页面卸载时正确 dispose 所有 Three.js 资源

## 10. 核心模块

| 模块 | 职责 |
|------|------|
| **DataStore** | IndexedDB 封装，管理学生数据和记录，降级到 localStorage |
| **EarthRenderer** | Three.js 3D 地球渲染，程序生成纹理 |
| **RandomPicker** | 加权随机选择算法，权重上限 5 |
| **UIController** | 界面交互控制，状态机管理 |
| **CSVExporter** | CSV 导出功能 |
| **MockData** | 内置模拟数据初始化 |

## 11. 错误处理

| 场景 | 处理方式 |
|------|----------|
| IndexedDB 不可用 | 自动降级到 localStorage，显示提示 |
| 存储空间不足 | 显示错误提示，建议清理记录 |
| Three.js 初始化失败 | 显示错误提示，降级到纯 CSS 动画 |
| 无学生数据 | 不应发生（内置数据），显示错误 |
| 导出失败 | 显示具体错误信息 |

**Loading 状态**:
- 首次加载：显示"初始化中..."
- 数据库操作：不阻塞，异步处理

## 12. 浏览器兼容性

- 最低要求：Chrome 90+, Firefox 88+, Safari 14+
- 必需特性：IndexedDB, WebGL, ES6+

## 13. 模拟数据说明

系统首次运行时，自动将内置的 20 个模拟学生姓名初始化到数据库中。后续使用中，这些数据会根据点名记录持续更新（主要是 lateCount 字段）。

如需更换学生名单，可以通过"清空数据"后刷新页面重新初始化。
